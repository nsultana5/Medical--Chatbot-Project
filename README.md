### STEPS:

Clone the repository

'''bash
Project repp:https://github.com/
'''

'''bash
conda create -n medibot python=3.10 -y
'''

'''bash
conda activate medibot
'''

### STEP 02-install the requirements
'''
bash
pip install -r requirements.txt
'''
